Loop Principles

L1 Playwright real-browser smoke is the correctness signal — build/EV alone don't close the loop
L4 reviewer ≠ author (fresh context, diff only)
L6 one file / one logical unit per execute part
L7 externalize state — handoff docs, CONTEXT.md, BUGS/FEATURES/DECISIONS
L8 bounded correction budget, retry-on-red
L9 explicit stop at every human / tier-3 gate
